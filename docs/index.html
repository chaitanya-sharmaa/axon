How it works

One line. Zero code changes.

Change only base_url. Everything else — compression, caching, routing, and protection — happens automatically.

Before Axon 
import openai

client = openai.OpenAI(
    api_key="sk-..."
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user",
               "content": "Hello!"}]
)
VS
After Axon ↓ 70% cost 
import openai

client = openai.OpenAI(
    api_key="sk-...",
    base_url="http://localhost:8080/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user",
               "content": "Hello!"}]
)
# x-axon-metrics: {"savings_pct": 39.3, "cost_saved_usd": 0.0021}

Configuration

Token compression is always on. The rest is up to you.

Axon Bridge is a token-compression-first proxy. All compression features are enabled by default. Security, routing, and memory features are disabled by default and must be explicitly opted-in via your .env file.

Feature CategoryFeatureDefault State.env Variable to Enable

Token Compression L1 Exact-Match & L2 Semantic Cache ✅ Enabled AXON_ENABLE_EXACT_MATCH_CACHE, AXON_ENABLE_SEMANTIC_CACHE
Token Compression Agentic Pipeline (8 passes) ✅ Enabled AXON_ENABLE_AGENTIC_OPTIMIZATIONS
Token Compression Tool Call Compression ✅ Enabled AXON_ENABLE_TOOL_COMPRESSION

Agentic Pipeline

98% reduction on agentic token loops.

8 mathematical compression passes run before any structural compression, specifically designed for ReAct, Plan-Execute, and tool-calling agent workflows.

Pass 1

Error Truncation

Compresses 50-line stack traces to the single error headline.

~90% on failed tools
Pass 2

Whitespace Normalization

Strips invisible Unicode, normalizes line endings, collapses spacing.

5–15% savings
Pass 3

Scratchpad Compression

Deduplicates sentences and strips filler from <thinking> / ReAct blocks.

30–50% on reasoning
Pass 4

Parallel Deduplication

Removes duplicate fields across overlapping tool results in the same turn.

15–40% savings
Pass 5

Prefix Caching

Auto-injects provider-native cache_control on stable system prompts.

85–90% on prefixes
Pass 6

Schema Differential

Omits JSON schemas for tools the agent hasn't used recently.

80% on schemas
Pass 7

Observation Window

Shannon entropy × recency to prune old, low-information tool results.

40–70% on history
Pass 8

Loop Circuit Breaker

Detects duplicate tool calls with identical args. Returns cached result, bypasses LLM entirely.

100% LLM bypass!

Quick Start

Running in 60 seconds.

1

Install

pip install axon-bridge
2

Configure

cp .env.example .env
# Set OPENAI_API_KEY=your-key
3

Serve

axon serve
# Dashboard → http://localhost:8080/dashboard
4

Point your SDK at Axon

import openai
client = openai.OpenAI(
    base_url="http://localhost:8080/v1",
    api_key="any-value"  # Axon uses your .env key
)
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
# Token savings start immediately.

Or run with Docker:

docker-compose up
# Axon → http://localhost:8080 · Dashboard → /dashboard

Custom Headers

Unlock advanced features with one header.

X-Axon-Session-ID

Enable session memory, fact extraction, and the full agentic pipeline.

X-Axon-Stateful-Thread: true

Send only the new message. Axon rehydrates the full history from local storage — 99% bandwidth saved.

X-Axon-Tenant-ID

Identify your tenant for per-tenant USD quota tracking and enforcement.

X-Axon-Max-Spend: 0.05

Per-request USD budget. Axon kills the streaming connection if the agent exceeds it.
